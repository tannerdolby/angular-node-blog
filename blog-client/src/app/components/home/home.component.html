Angular + Node.js Blog Template

This is an Angular and Node.js blog template. The blog posts are created within the /pages directory and are rendered on their respective pages using an API endpoint and a single "posts" component which acts as a template to display each blog post. Update the /assets/blog.json data with each blog posts metadata.
